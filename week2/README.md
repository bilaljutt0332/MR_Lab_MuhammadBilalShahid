# Week 2: ROS 2 Topics and Services
This folder contains the deliverables for Lab Manual 2. 

### Quick Notes:
* This lab focused on interacting with the **Turtlesim** package using CLI tools and RQT.
* No custom packages were created in this session as the focus was on understanding **Topics** (continuous data) and **Services** (on-demand requests).
* Independent control of a second turtle was achieved by targeting the specific namespace topic `/turtle2/cmd_vel`.
